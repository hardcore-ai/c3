# Hardcore AI by 30X — Cohorte 3

Repositorio de materiales y entregables de la **Cohorte 3** de Hardcore AI.

El programa es un journey de 12 sesiones (11 estaciones + Demo Day) donde construyes un producto real de punta a punta con AI, semana a semana, hasta el Demo Day (**lunes 13 de julio de 2026**).

## Contenido

| Carpeta | Qué es |
|---------|--------|
| [`Estación 1`](./Estación%201) | Introducción y Demostración — templates de la tarea (Product Vision Board / Internal Solution Brief), banco de productos y la demo de referencia AI-DLC |
| [`Estación 2`](./Estación%202) | Descubrimiento y Definición — plantilla para co-crear tu PRD con AI: `prompts-especificacion.md`, ejemplo trabajado (AgentVault) en `docs/` y `specs/` para tu entregable |
| [`Estación 3`](./Estación%203) | Ingeniería de Software Agéntica — clase con Claude Code, OpenCode y Antigravity: `clase-3-estudiante.md`, `codelab_page/` y `workshops/` |
| [`Estación 4`](./Estación%204) | Diseñando el QUÉ con AI-DLC (fase Inception) — framework AI-DLC, recursos de apoyo y proyecto de ejemplo con los artefactos de Inception generados |
| [`Estación 5`](./Estación%205) | Diseñando el CÓMO con AI-DLC (fase Construction) — artefactos de Construction de la primera unidad, `aidlc-rules/` y `estacion5-runbook.md` |
| [`Estación 6`](./Estación%206) | Implementando — Agentic Engineering más allá de AI-DLC: scaffolding operativo, taxonomía de modelos/arneses/orquestación, `estacion6-runbook.md`, `prompts.md` y slides (HTML/PDF) |

> Las estaciones siguientes se irán agregando a medida que avanza la cohorte.
